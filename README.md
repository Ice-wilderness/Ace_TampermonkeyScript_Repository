## 通用安装方式

1. 安装 Tampermonkey、Violentmonkey 或其他兼容的用户脚本管理器。
2. 在脚本管理器中新建脚本。
3. 将对应 `.js` 文件内容粘贴进去并保存。
4. 打开匹配网站后脚本会自动运行。

## Bilibili 视频观看历史记录

文件：[Bilibili视频观看历史记录.js](./Bilibili视频观看历史记录.js)

用于记录 Bilibili 视频访问和观看进度，并在首页、搜索页、历史页、稍后再看、视频推荐、合集/分 P 列表等位置标记已观看内容。

### 主要功能

- 自动记录视频访问状态和观看进度。
- 在视频封面上显示已访问、已观看、观看百分比或多 P 记录提示。
- 支持普通分 P 视频、合集视频、合集变种页面和播放器选集列表。
- 支持进度条显示、标签位置、标签透明度、低/高进度阈值配置。
- 支持续播提示，可跳转到上次观看进度。
- 内置历史管理面板，支持搜索、筛选、排序、分页、删除、导入、导出。
- 采用分片存储，减少大量历史记录时的读写压力。
- 提供 localStorage 临时备份恢复，降低异常退出时丢失进度的概率。

### 历史管理

- 按标题、BV、av 搜索。
- 按已观看、已访问筛选。
- 按时间、进度、标题排序。
- 分页浏览历史记录。
- 删除单条或批量删除记录，删除后可短时间内撤销。
- 导入、导出 JSON 格式历史记录。

### 数据说明

脚本使用 `GM_setValue` / `GM_getValue` 保存数据，并将记录拆分到多个分片键中。历史记录 key 通常为：

- `BVxxxxxxxxxx`
- `BVxxxxxxxxxx?p=2`
- `av123456`

普通单 P 视频记录为基础 key，多 P 视频会按分 P 保存为 `?p=` key。

### 配置项

- 是否显示进度条。
- 是否显示已访问标记。
- 是否显示自动续播提示。
- 是否开启调试日志。
- 标签透明度。
- 标签位置。
- 低进度和高进度阈值。

### 兼容页面

- 视频页。
- 首页和带查询参数的首页。
- 动态页。
- 空间页。
- 历史页。
- 稍后再看。
- 搜索页。
- 合集、播放列表相关页面。
